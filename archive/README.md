# Archived Code

This directory contains the original exploratory code from the `did_designs` repository before the restructuring to the "Forward-Engineering" approach.

## Contents

### Notebooks (`notebooks/`)

- **`2x2.ipynb`** - DiD methodology exploration with pyfixest and statsmodels
- **`beer.ipynb`** - Fixed effects panel analysis (beer taxes and traffic fatalities)
- **`test.ipynb`** - Comprehensive econometrics methods tutorial
- **`temp.ipynb`** - Bibliography deduplication utility

### Scripts (`scripts/`)

- **`data.R`** - Master data download script for 11+ panel/DiD datasets
- **`medicaid.R`** - Medicaid expansion 2x2 DiD analysis replication

### Bibliography

- **`input.bib`** - Original bibliography with duplicates
- **`output.bib`** - Deduplicated bibliography

## Note

This code was written during initial exploration and learning. The main repository now follows a structured "Forward-Engineering" approach based on Baker, Callaway, Cunningham, Goodman-Bacon, and Sant'Anna (2025).
